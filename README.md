---------------------------------------------------------------------
1. PREPARACIÓN DE TU CUENTA (BITSO ALPHA)
---------------------------------------------------------------------
- Entra a tu portal de Bitso, ve a la sección de API y genera tus llaves.
- Activa únicamente los permisos de:
    * "Consulta de saldos / Lectura"
    * "Colocar órdenes / Trading"
- IMPORTANTE: Mantén desactivado el permiso de "Retiros externos". 
  Esto garantiza que tus fondos estén blindados en todo momento.
- Asegúrate de tener al menos $500.00 MXN en tu cuenta para iniciar.

---------------------------------------------------------------------
2. EJECUCIÓN POR PRIMERA VEZ (WINDOWS DEFENDER)
---------------------------------------------------------------------
Al ser un software independiente de trading, Windows Defender podría 
mostrar una alerta de protección al ejecutarlo por primera vez:

1. Haz doble clic en "SynapseQuantSuite.exe".
2. Si aparece la ventana azul "Windows protegió su PC":
   - Haz clic en el enlace "Más información".
   - Aparecerá un botón nuevo: "Ejecutar de todas formas". Haz clic ahí.
3. El sistema operativo registrará esta excepción y las siguientes 
   veces podrás abrir la aplicación con un doble clic normal.

*CONSEJO PRO: Si notas que la app no guarda tus configuraciones, 
haz clic derecho sobre el .exe y selecciona "Ejecutar como administrador".*

---------------------------------------------------------------------
3. MODELO DE NEGOCIO (100% ÉTICO - DEVFEE 10%)
---------------------------------------------------------------------
Synapse utiliza un modelo de éxito compartido:
- El software es gratuito.

  un 10% del profit como comisión de desarrollo y transfiere el 90% 
  restante a tu cofre de USDT en Bitso para proteger tu capital.
- Si el mercado no genera ganancias, el servicio no te cobra nada.
